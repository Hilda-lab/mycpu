# myCPU 项目状态与开发进度

## 📊 当前项目状态

#### 1. CPU 核心模块 ✅
- 8 个 32 位通用寄存器（EAX, EBX, ECX, EDX, ESI, EDI, EBP, ESP）

- 主循环：取指→译码→执行
- 中断支持（IF标志，中断上下文保存）

#### 3. 内存系统 ✅
- 16 MB 平坦内存空间
- 边界检查和异常处理
- 8/16/32 位读写支持
- 小端序字节序

#### 4. 中断系统 ✅ (新增)

**PIC 中断控制器**
- 支持 16 个 IRQ（IRQ0-IRQ15）
- 中断优先级管理
- 中断屏蔽（IMR）
- 中断请求寄存器（IRR）
- 中断服务寄存器（ISR）
- EOI（中断结束）处理

**PIT 定时器**
- 标准 8254 实现
- 3 个独立通道
- 多种工作模式（模式0-5）
- 可编程频率
- 定时器中断触发

**UART 串口**
- 完整的 16550 兼容实现
- 发送/接收 FIFO
- 中断支持（接收、发送、线路状态）
- 波特率配置
- 线路控制和Modem控制

**IDT 中断描述符表**
- 支持 256 个中断向量
- 中断门和陷阱门
- 异常向量（0-19）
- 用户中断向量（32-255）

#### 5. 内存管理 ✅ (新增)

**MMU 内存管理单元**
- 虚拟地址翻译
- 二级页表结构
- 4KB 页面大小
- 页表项标志（存在、读写、用户等）

**TLB 转换后备缓冲区**
- 64 项 TLB 缓存
- TLB 命中/未命中统计
- TLB 刷新

**分页功能**
- CR0.PG 分页使能
- CR3 页目录基址
- CR2 页错误地址
- 恒等映射支持
- 页错误异常

#### 6. 汇编器 ✅
- 支持完整指令集汇编
- 标签和跳转目标解析
- 注释支持（`;` 和 `#`）
- 立即运行选项（`-r`）
- 机器码输出（`-d`）

#### 7. 调试器 ✅
- ncurses 可视化界面
- 单步执行和连续运行
- 断点设置和管理
- 寄存器和标志位监视
- 内存查看和反汇编

#### 8. 测试系统 ✅
- 自动化测试框架（CTest）
- 新指令专项测试（test_new_inst）
- 中断和分页测试（test_interrupt）
- Fibonacci 完整测试
- 汇编程序测试用例
